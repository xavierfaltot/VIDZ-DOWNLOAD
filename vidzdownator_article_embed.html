À quoi sert VIDZDOWNATOR ?

VIDZDOWNATOR est un petit boîtier logiciel dédié à une seule opération : récupérer des vidéos depuis des liens, les nommer proprement, extraire leurs métadonnées et les déposer dans un dossier d’import. Il ne remplace pas un logiciel de montage. Il agit avant le montage, comme une machine d’ingest.

L’utilisateur colle une ou plusieurs URLs, par exemple des reels Instagram, des vidéos YouTube ou des liens Vimeo. Le système détecte la source automatiquement. Il n’y a pas de menu compliqué : START lance le download, FOLDER ouvre le dossier, STOP interrompt la tâche.

Comment ça marche ?

  • VIDZDOWNATOR lit les URLs collées dans le champ d’entrée.
  • Il détecte la source depuis l’adresse : Instagram, YouTube, Vimeo ou web générique.
  • Il télécharge chaque vidéo avec son titre, sa chaîne, sa date et sa miniature quand elles sont disponibles.
  • Il fabrique un nom de fichier lisible avec artiste, collection, mots-clés et titre.
  • Il range tout dans le dossier VIDZ IMPORTS, prêt pour un workflow de montage.

Pourquoi cette forme ?

Le projet a été pensé comme une machine industrielle plutôt qu’une fenêtre d’application classique : façade noire, écrans ambrés, boutons physiques, LED de progression. L’idée est de donner une fonction précise à chaque outil, comme dans une régie broadcast ou une gamme d’instruments spécialisés.

Comment il a été fabriqué

VIDZDOWNATOR a été construit avec Codex comme compagnon de fabrication : conception de l’interface, prototypage Python, logique de téléchargement, création d’une app macOS locale, préparation du dépôt GitHub et documentation. Codex a servi d’atelier logiciel : on a avancé par essais, captures, corrections et décisions de design jusqu’à obtenir une machine partageable.

What is VIDZDOWNATOR for?

VIDZDOWNATOR is a small software box dedicated to one operation: collecting videos from links, naming them cleanly, extracting metadata and placing them into an import folder. It does not replace editing software. It works before editing, like an ingest machine.

The user pastes one or many URLs, such as Instagram reels, YouTube videos or Vimeo links. The system detects the source automatically. There is no complicated menu: START downloads, FOLDER opens the destination folder, STOP interrupts the task.

How does it work?

  • VIDZDOWNATOR reads the URLs pasted into the input field.
  • It detects the source from the address: Instagram, YouTube, Vimeo or generic web.
  • It downloads each video with title, channel, date and thumbnail when available.
  • It creates readable filenames with artist, collection, keywords and title.
  • It stores everything in VIDZ IMPORTS, ready for an editing workflow.

Why this shape?

The project was designed as an industrial machine rather than a normal app window: black metal faceplate, amber screens, physical buttons and progress LEDs. The idea is to give every tool one precise job, like equipment in a broadcast control room or a family of specialised instruments.

How it was made

VIDZDOWNATOR was built with Codex as a fabrication partner: interface design, Python prototyping, download logic, creation of a local macOS app, GitHub packaging and documentation. Codex acted as a software workshop: we moved through tests, screenshots, corrections and design decisions until the machine became shareable.
