FR · Article en français

À quoi sert VIDZDOWNLOAD ?

VIDZDOWNLOAD est un petit boîtier logiciel dédié à une seule opération : récupérer des vidéos depuis des liens, les nommer proprement, extraire leurs métadonnées et les déposer dans le dossier choisi par l’utilisateur. Il ne remplace pas un logiciel de montage. Il agit avant le montage, comme une machine d’ingest.

L’utilisateur colle une ou plusieurs URLs, par exemple des reels Instagram, des posts carrousel, des vidéos YouTube ou des liens Vimeo. Il peut aussi indiquer un fichier export de bookmarks ou un dossier contenant des bookmarks. Le système extrait les URLs, détecte la source automatiquement et dépose les vidéos dans le dossier final paramétré.

Comment ça marche ?

  • VIDZDOWNLOAD lit les URLs collées dans le champ d’entrée.
  • Il peut aussi lire un export de bookmarks ou un dossier contenant des fichiers `.html`, `.json`, `.txt`, `.url` ou `.webloc`.
  • Il détecte la source depuis l’adresse : Instagram, YouTube, Vimeo ou web générique.
  • Il télécharge chaque vidéo avec son titre, sa chaîne, sa date et sa miniature quand elles sont disponibles.
  • Quand une URL Instagram contient un carrousel, il ajoute les vidéos disponibles une par une à la file de téléchargement.
  • Il fabrique un nom de fichier lisible avec artiste, collection, mots-clés et titre.
  • Il range tout dans le dossier final choisi dans OUTPUT FOLDER, par défaut VIDZ IMPORTS.

Pourquoi cette forme ?

Le projet a été pensé comme une machine industrielle plutôt qu’une fenêtre d’application classique : façade noire, écrans ambrés, boutons physiques, LED de progression. L’idée est de donner une fonction précise à chaque outil, comme dans une régie broadcast ou une gamme d’instruments spécialisés.

Comment il a été fabriqué

VIDZDOWNLOAD a été construit avec Codex comme compagnon de fabrication : conception de l’interface, prototypage Python, logique de téléchargement, création d’une app macOS locale, préparation du dépôt GitHub et documentation. Codex a servi d’atelier logiciel : on a avancé par essais, captures, corrections et décisions de design jusqu’à obtenir une machine partageable.

EN · Article in English

What is VIDZDOWNLOAD for?

VIDZDOWNLOAD is a small software box dedicated to one operation: collecting videos from links, naming them cleanly, extracting metadata and placing them into the output folder chosen by the user. It does not replace editing software. It works before editing, like an ingest machine.

The user pastes one or many URLs, such as Instagram reels, carousel posts, YouTube videos or Vimeo links. They can also point the app to a bookmark export file or a folder containing bookmarks. The system extracts URLs, detects the source automatically and stores videos in the configured final folder.

How does it work?

  • VIDZDOWNLOAD reads the URLs pasted into the input field.
  • It can also read a bookmark export or a folder containing `.html`, `.json`, `.txt`, `.url` or `.webloc` files.
  • It detects the source from the address: Instagram, YouTube, Vimeo or generic web.
  • It downloads each video with title, channel, date and thumbnail when available.
  • When an Instagram URL contains a carousel, it queues the available videos one by one.
  • It creates readable filenames with artist, collection, keywords and title.
  • It stores everything in the final folder chosen in OUTPUT FOLDER, defaulting to VIDZ IMPORTS.

Why this shape?

The project was designed as an industrial machine rather than a normal app window: black metal faceplate, amber screens, physical buttons and progress LEDs. The idea is to give every tool one precise job, like equipment in a broadcast control room or a family of specialised instruments.

How it was made

VIDZDOWNLOAD was built with Codex as a fabrication partner: interface design, Python prototyping, download logic, creation of a local macOS app, GitHub packaging and documentation. Codex acted as a software workshop: we moved through tests, screenshots, corrections and design decisions until the machine became shareable.
